### 1️⃣ Clone the Repository
```sh
git clone https://github.com/NikkiAung/React-Native-To-Do-App.git
cd React-Native-To-Do-App
```

### 2️⃣ Install Dependencies
```sh
npm install
```

### 3️⃣ Start the Development Server
```sh
npm start
```
